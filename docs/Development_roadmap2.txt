Features

✓ Search by Username

✓ User Cards

✓ Open Public Profile

✓ Infinite Scrolling

# Version 1 Exclusions

The following ideas are intentionally excluded from Version 1.

* Friends
* Followers
* Notifications
* Comments
* Likes
* Activity Feed
* Steam Integration
* Achievement Sync
* Playtime Sync
* Game Recommendations
* AI Features
* Journal Images
* Seasonal Events
* Social Messaging

These ideas remain documented for future versions.
