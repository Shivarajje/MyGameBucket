# Table 8 — manual_game_submissions

## Purpose

Stores games submitted by users that are not found through the external Game API.

These submissions remain private until reviewed by an administrator.

Approved submissions become part of the shared `game_catalog`.

# Workflow

User submits game

↓

Status = Pending

↓

Administrator reviews

↓

Approved

↓

Game inserted into `game_catalog`

OR

↓

Rejected

↓

Reason stored
